关于本站

本博由Hexo动力系统搭建，内容托管在七牛云

为什么一直坚持独立建站？

2011年开始接触写博客，当时由WordPress构建在AWS，后来免费一年的EC2到期，转站个大云厂商（AWS->Azure->Aliyun->SAE->BAE->腾讯云->Qingcloud->GCP->首都在线）；后来换 Solo - Java 博客系统+Docker放在Daocloud上 后来Daocloud不再提供免费容器托管告终。期间多次搬家，服务器磁盘故障数据全部丢失。现在从头开始，不知道能坚持多久，但是我会尽力坚持下去。

关于写博客网上有很多平台可以选择：51cto、CSDN、简书、QQ空间、微信公众号等，但是这么多年一直坚持独立建站，独立服务器、虚拟空间、vps、各种云服务器、docker托管平台、代码平台各种折腾。
之所以选择独立建站是对建站技术有个更全面的了解；系统架构、SEO、站点安全、政策合规等、虽然在折腾的路上磕磕碰碰但是收获颇丰（因为个云厂商的羊毛有限而自己又舍不得掏钱、不得不兜兜转转；因为wordpress老是被黑所以久病成良医；因为找不到自己喜欢的主题模板所以自己动手写）
啰嗦那么多其实就是为了做一个有逼格的技术宅，嗯现在Hexo+Markdown方式很适合我，不折腾了。

本站不接受任何广告赞助，即使只有我一个用户。
